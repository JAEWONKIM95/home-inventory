# screenshots 폴더

여기에 앱 화면 캡처를 넣으면, 메인 README 상단의 표에 자동으로 표시됩니다.

## 넣을 파일 이름
- `1-map.png` — 전체 물건 지도(평면도)
- `2-room.png` — 방 내부(가구 배치)
- `3-calendar.png` — 달력 화면

## 캡처 방법
1. 앱을 브라우저에서 엽니다 (배포 주소 또는 `http://localhost:5173`)
2. `Win + Shift + S`(Windows) 또는 `Cmd + Shift + 4`(Mac) 로 영역 캡처
3. 위 이름의 PNG로 저장해서 이 폴더에 넣기

> 🙈 공개 저장소에는 실제 물건 이름이 보일 수 있습니다. 샘플 화면으로 안전하게 찍으려면:
> **"내보내기"(백업) → "초기화"(샘플 표시) → 캡처 → "가져오기"(복원)** 순서로 하세요.
> ⚠️ "초기화"는 현재 데이터를 샘플로 **덮어쓰니** 반드시 먼저 백업하세요.
